A fun and interactive **Rock Paper Scissors** game built using Python. Challenge the computer, test your luck, and enjoy a classic game right from your terminal!

---

## 📌 About the Project

This project was developed as **Task 3** of the **CodeSoft Python Internship**.

The game allows the user to play Rock, Paper, or Scissors against the computer. The computer randomly selects its move, and the game determines whether the player wins, loses, or ties. Players can continue playing multiple rounds until they decide to exit.

## 🚀 How to Run

### Step 1:
Clone the repository

```bash
git clone <repository-link>
```

### Step 2:
Navigate to the project folder

```bash
cd Rock_Paper_Scissors
```

### Step 3:
Run the Python file

```bash
python rock_paper_scissors.py
```
